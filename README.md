# Shadow Creation Tool

Built with React, this tool lets users adjust settings and create a variety of shadows. It displays the generated code in a box so users can directly copy it into their own projects. The project is designed with a focus on usability and visual design.



# ابزار ساخت سایه ها


با استفاده از ری‌اکت ساخته شده، که به کاربران امکان تغییر تنظیمات و ایجاد سایه‌های متنوع را می‌دهد. این ابزار کد تولید شده را درون باکس نمایش می‌دهد تا کاربران بتوانند آن را مستقیماً در پروژه‌های خود استفاده کنند. این پروژه با تمرکز بر کاربردپذیری و طراحی بصری ایجاد شده است.
